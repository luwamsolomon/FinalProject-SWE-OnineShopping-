Welcome to Online Shopping

This is a demo Spring Boot WebApp, for CS425-SWE Final Exam
It features a basic demonstration of the following technologies for developing an Enterprise Application, on the Spring and Java stack:
  • Spring Web MVC with Thymeleaf View Template Engine
  • Spring Data - Java Persistence API (JPA) with Hibernate
  • Spring REST for providing RESTful Web Service endpoints
  • JQuery AJAX and JSON
  • Bootstrap 4 for CSS styling
  • MySQL or Apache Derby Database
